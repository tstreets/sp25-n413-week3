== Description ==

SmileX is a modern, responsive WordPress block theme specifically designed for dental clinics and healthcare professionals. It features full-site editing, customizable templates, and a clean, professional design tailored for dental practices, clinics, and healthcare service providers. SmileX allows you to easily adjust color schemes, typography, and layout styles with WordPress's latest tools. Fully responsive and translation-ready, SmileX ensures an exceptional user experience on any device.
